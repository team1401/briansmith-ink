# Brian Smith - Author Website

Professional author website for Brian Smith, featuring his book *Aluminum & Dust*, Substack newsletter *Nomad's Edge*, and Radio Terlingua project.

## Technology Stack

- Pure HTML5/CSS3
- Vanilla JavaScript (no frameworks)
- Mobile-responsive design
- Google Fonts (Crimson Pro + Inter)
- GitHub Pages hosting

## Design

- Dark/moody desert aesthetic
- Typography-first layout
- Warm sand, burnt orange, and charcoal color palette
- Smooth scroll navigation
- Fade-in animations
- Premium feel with generous whitespace

## Sections

1. **Hero** - Name, tagline, and CTAs
2. **About** - Author bio and current projects
3. **Books** - Featured book with Amazon links + works in progress
4. **Writing** - Substack newsletter subscription
5. **Projects** - Radio Terlingua
6. **Contact** - Social links

## Deployment

Site is deployed via GitHub Pages at: https://[username].github.io/briansmith-ink/

Custom domain: briansmith.ink (requires DNS configuration)

## Local Development

Simply open `index.html` in a browser. No build process required.
